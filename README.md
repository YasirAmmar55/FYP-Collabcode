# CollabCode 🚀

A real-time collaborative coding platform where developers can code, chat, and communicate in real time.

---

## ⚙️ Features
- 🧑‍💻 Real-time code editing
- 💬 Live chat system
- 🎥 Video/audio calling (WebRTC)
- 🔄 Instant sync using Socket.io
- 👥 Multi-user rooms

---

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- Socket.io
- WebRTC
- MongoDB
